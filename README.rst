My dotfiles, managed by `polkadots`
===================================

To use:
-------

Install polkadots
~~~~~~~~~~~~~~~~~

`pip install polkadots_dotfile_manager`

Install the dotfiles
~~~~~~~~~~~~~~~~~~~~
.. code:: bash

        # go meta
        ln -s "$(pwd)/polkadots" ~/.config/polkadots
        # deal with the rest of them
        polkadots
